NAME

geopm_comm - geopm mpi communicator manipulation

SYNOPSIS

#include <geopm.h>

Link with -lgeopm

int geopm_comm_num_node(

MPI_Comm comm,
int *num_node);

int geopm_comm_split(

MPI_Comm comm,
MPI_Comm *split_comm,
int *is_ctl_comm);

int geopm_comm_split_ppn1(

MPI_Comm comm,
MPI_Comm *ppn1_comm);

int geopm_comm_split_shared(

MPI_Comm comm,
MPI_Comm *split_comm);

DESCRIPTION

Interfaces for MPI communicator use by geopm.

geopm_comm_num_node()

sets num_node to the number of compute nodes associated with the communicator comm. A compute node is defined to be a shared memory coherency domain.

geopm_comm_split()

splits the input communicator comm into two: one which is comprised of the the lowest rank on each compute node (the controller communicator), and the other comprised of the rest (the application communicator). The split_comm will is the former if is_ctl_comm is non-zero and the the latter otherwise.

geopm_comm_split_ppn1()

is the same as geopm_comm_split() except that the output split communicator ppn1_comm is set to MPI_COMM_NULL unless the calling process is the lowest rank on the node then it will be set to the controller communicator.

geopm_comm_split_shared()

creates a new communicator split_comm which is comprised of all of the ranks that share the same compute node as the calling process and are a member of the input communicator comm.

ERRORS

All functions described on this man page return an error code. See geopm_error(3) for a full description of the error numbers and how to convert them to strings.
